# Ansta Easyjak React

Ansta Easyjak React is a fan-made open-source utility for Ensemble Stars!! Music players.  
It helps users search songs and estimate the starting combo point needed to clear Easy difficulty with a small safety margin.

This project is a React-based rewrite of the original HTML/JavaScript version, with improved maintainability, multilingual support, mobile usability, and progressive web app support.

## Features

- Song search by Korean, Japanese, and English titles
- Easy clear prediction based on song note data and measured values
- Multilingual UI: Korean, Japanese, and English
- Mobile-friendly responsive layout
- Song list filtering and sorting
- YouTube reference video links for measured songs
- PWA support for app-like usage on mobile devices
- Data synchronization from the original song data source
- Separation of prediction logic into reusable pure functions

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- GitHub Pages
- PWA manifest and service worker

## Project Purpose

The goal of this project is to provide a lightweight and accessible tool for players who want to plan Easy difficulty clears more efficiently.

The project also serves as an open-source frontend/data utility project focused on:

- refactoring a legacy HTML/JavaScript tool into React
- maintaining structured song data
- improving multilingual search
- validating prediction logic
- improving mobile user experience
- making the tool easier to extend and maintain

## Current Status

The React version is under active development.  
The original HTML version remains available as the stable public version, while the React version is being improved as the next-generation version.
